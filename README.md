# Bus Reservation System

A console-based Bus Reservation System developed using Java, JDBC, and MySQL.

## Features
- Book bus tickets
- Store reservation details in MySQL
- JDBC database connectivity
- DAO design pattern
- Object-Oriented Programming concepts

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Project Structure
- Bus.java
- Booking.java
- BusDAO.java
- BookingDAO.java
- DbConnection.java
- BusReservation.java
